### Предварительные требования

1. Установите Node.js 
2. Установите Git
3. Установите приложение Expo Go на телефон:


### Запуск проекта  

1. Клонируйте репозиторий:

   cd название-репозитория
2. Установите зависимости:
    ```bash
    npm install
3. Запустите проект
    ```bash
    npx expo start
